# Prospection Dentistes — DreamNova Voice

## Critères de Qualification (TOUS doivent matcher)

1. **Téléphone très visible** sur site/Google Maps (business phone-driven)
2. **Pas de booking en ligne** (pas Doctolib, ZocDoc, ou système de RDV intégré)
3. **Plusieurs praticiens** (flux d'appels élevé, secrétaire débordée)
4. **Avis négatifs mentionnant** : attente, non-réponse, réception débordée
5. **Site web faible ou ancien** (pas de concurrent IA en place)
6. **Horaires tendus** : petite équipe, forte pression d'accueil

## Mots-clés dans les Avis Google (Hébreu)

- לא עונים / לא ענו לטלפון
- המתנה ארוכה / חיכיתי הרבה
- קשה לקבוע תור / קשה להשיג תור
- שירות גרוע אבל הרופא מעולה
- המזכירה לא ענתה / הקו תפוס
- ביטלו לי תור בלי להודיע

## Mots-clés dans les Avis Google (Français)

- impossible de les joindre / ne répond pas
- attente interminable / longue attente
- difficile d'avoir un rendez-vous
- secrétaire débordée / accueil nul
- on m'a pas rappelé

```csv
"Dr. [NOM]","+972-XX-XXX-XXXX","https://...","4.2","לא עונים לטלפון - 3 ביקורות","ירושלים","3","Non","HOT"
```

**Statuts :**
- **HOT** = Tous les critères matchent + avis négatifs sur le téléphone
- **WARM** = 4/6 critères + pas d'avis négatifs explicites
- **COLD** = Moins de 4 critères ou booking online existant

## Process de Qualification (Étape par Étape)

1. Ouvrir Google Maps → chercher "מרפאת שיניים" ou "cabinet dentaire [VILLE]"
2. Filtrer : note 3.5-4.5 (les 5.0 ont moins de problèmes, les <3.5 ont d'autres soucis)
3. Lire les avis 1-3 étoiles → chercher les mots-clés ci-dessus
4. Vérifier le site web → booking online ? site moderne ?
5. Compter les praticiens listés (Google ou site)
6. Si 4+ critères matchent → ajouter au CSV comme HOT
7. Préparer le message WhatsApp personnalisé (mentionner un avis spécifique si possible)
8. Envoyer avec le lien Loom

## Horaires d'Envoi Optimaux

- **Israël** : Dimanche-Jeudi, 10h-12h ou 14h-16h (pas le matin tôt — occupation téléphonique max)
- **France** : Mardi-Jeudi, 11h-13h ou 16h-18h (éviter lundi matin et vendredi après-midi)
- **JAMAIS** : Shabbat, mardi soir (David), jours fériés
